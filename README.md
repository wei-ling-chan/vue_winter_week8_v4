# 麵鋪子
一個以Vue3 + Pinia 製作的麵店形象官網
![首頁](https://i.imgur.com/Lhm5pPE.png)

## Demo

https://wei-ling-chan.github.io/vue_winter_week8_v4/#/

## 簡介

由於朋友開了一間麵店，因此作為靈感，作為 Vue3 前端技術練習，其主要功能如下：

-前台:
1.首頁
2.關於我們
3.美味專區
4.最新消息
5.我的收藏
6.購物車

-後台:
1.產品管理(新增、編輯、刪除)
2.訂單管理(檢視、刪除)
3.優惠卷管理(新增、編輯、刪除)

## 使用技術

* Vue3+vite
* VueRouter
* Pinia
* JavaScript ES6* 
* Bootstrap

## 使用 Plugin

* bootstrap-icons
* sweetalert2
* swiper
* v-calendar
* Vee-validate
* vue-loading-overlay
* vue-axios
* Eslint
* prettier
